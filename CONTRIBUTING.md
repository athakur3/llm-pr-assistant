# Contributing

## Setup

```bash
npm install
npm run compile
```

## Build and package

```bash
npm run build
npm run package
```

## Publish

```bash
npm run publish:vscode
npm run publish:ovsx
```

## Notes

- `./rebuild.sh` runs a clean build and packages a VSIX.
- Ensure `OPEN_VSX_TOKEN` and VS Code Marketplace credentials are set before publishing.
